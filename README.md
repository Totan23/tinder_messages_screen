# Tinder Messages Screen

Proyecto iOS en Swift que replica la pantalla de mensajes de Tinder. Incluye una vista de coincidencias recientes con imágenes circulares y una lista de conversaciones con foto de perfil, nombre de usuario y último mensaje. Utiliza LBTATools para una construcción rápida de interfaces y layouts programáticos sin storyboards.
